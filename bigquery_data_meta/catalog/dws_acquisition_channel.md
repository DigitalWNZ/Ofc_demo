# Acquisition Channel Performance

User acquisition channel performance metrics aggregated daily. Tracks installs, costs, revenue, and ROI by acquisition source, medium, and campaign. Used for marketing ROI analysis, budget allocation optimization, and identifying the most cost-effective channels for player acquisition.

## Schema

| Column | Type | Description |
|--------|------|-------------|
| date | STRING | The reporting date in YYYYMMDD format. Partition key for tracking acquisition channel performance over time. |
| source | STRING | The acquisition source (e.g., google, facebook, apple_search_ads, organic). Identifies where the player was acquired from. |
| medium | STRING | The acquisition medium (e.g., cpc, cpi, organic, referral). Combined with source for full channel attribution. |
| campaign | STRING | The specific marketing campaign name or identifier. Enables campaign-level ROI analysis and creative performance evaluation. |
| installs | BIGINT | The number of new app installs attributed to this source, medium, and campaign on this date. |
| cost_usd | DOUBLE | The total marketing spend in USD for this acquisition channel on this date. Sourced from ad network cost data integrations. |
| revenue_usd | DOUBLE | The total revenue in USD generated by players acquired through this channel on this date. Used for ROI calculation. |
| roi | DOUBLE | Return on investment — (revenue_usd - cost_usd) / cost_usd. A negative value indicates the channel is not yet profitable. |
| d1_retention_rate | DOUBLE | Day-1 retention rate for players acquired through this channel. Indicates acquisition quality beyond volume. |
| d7_retention_rate | DOUBLE | Day-7 retention rate for players acquired through this channel. A stronger signal of player quality than D1 retention. |
| cost_per_install | DOUBLE | Cost per install (CPI) — cost_usd divided by installs. The primary efficiency metric for user acquisition budget management. |
